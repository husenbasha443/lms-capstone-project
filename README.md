## LTC AI Learning & Knowledge Intelligence Platform

This repository contains a production-grade, Azure-ready AI-enabled Learning & Knowledge Intelligence Platform (`ltc-ai-platform`) with:

- **Frontend**: React (Vite), JWT auth, protected routes, AI audio player
- **Backend**: FastAPI (async), PostgreSQL, SQLAlchemy, ChromaDB (persistent), Azure integrations
- **AI Layer**: CrewAI-based multi-agent pipelines for knowledge processing, RAG Q&A, and personalized revision

Detailed setup, configuration, and run instructions are provided after all code files are generated.
